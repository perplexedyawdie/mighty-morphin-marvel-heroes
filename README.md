## Mighty Morphin Marvel Heroes

Requirements
-------------
```
virtualenv
numpy
scikit_image
opencv_python
Pillow
cmake
dlib
```

Getting Started
-------------

#### Installation

```
npm i
pip3 install virtualenv
cd mighty-morphin-marvel-heroes/face-morpher
mkdir env
cd env
python3 -m venv env
source env/bin/activate
pip3 install numpy scikit_image opencv_python Pillow cmake dlib
```
